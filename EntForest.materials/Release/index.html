A work of interactive fiction.

EntForest was created with Inform and has IFID 4FA146B4-7A26-461F-A5B5-2F9F4A1B9CD7. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
